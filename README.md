# 🚀 Projet Node.js Débutant – Serveur HTTP & API Express

## 📌 Présentation
Ce projet est une introduction complète à Node.js. Il permet de comprendre comment créer un serveur HTTP simple, manipuler des fichiers, construire une API REST et utiliser Express.js pour structurer une application plus avancée.

L'objectif est de passer progressivement d’un serveur basique à une application backend moderne.

---

## ⚙️ Fonctionnalités
- Serveur HTTP simple avec Node.js
- Lecture de fichiers avec le module fs
- Création d'une API REST sans framework
- Utilisation d’Express.js
- Stockage de données en JSON
- Gestion des variables d’environnement (.env)
- Logs des requêtes avec Morgan
- Interface frontend simple en HTML
- Ajout et récupération des utilisateurs

---

## 📁 Structure

- server.js : serveur HTTP simple  
- app.js : application Express  
- users.json : données des utilisateurs  
- data.txt : fichier exemple  
- .env : variables d’environnement  
- package.json : configuration du projet  
- public/ : fichiers frontend  
  - index.html : interface utilisateur  

---

## 🧠 Objectifs pédagogiques
Ce projet permet d’apprendre :
- Le fonctionnement des requêtes HTTP
- La manipulation de fichiers avec Node.js
- La création d’une API REST
- L’intérêt d’un framework comme Express.js
- La communication entre frontend et backend

---

## 🌐 Routes de l'API

- `/` → Page d'accueil  
- `/api/users` → Récupérer tous les utilisateurs  
- `/api/users/:id` → Récupérer un utilisateur spécifique  
- `POST /api/users` → Ajouter un utilisateur  

---

## 🖥️ Frontend
Une interface HTML simple permet de :
- Charger les utilisateurs
- Ajouter un nouvel utilisateur
